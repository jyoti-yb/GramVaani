7️⃣ How Data Flows
User Query: "What pests are affecting maize in my area?"
    ↓
1. Fetch user location (Bangalore, Karnataka)
    ↓
2. Query MongoDB:
   • Hyperlocal data (soil, crops)
   • Pest reports (Fall Armyworm in Maize)
   • Disease reports (if any)
   • Success stories (nearby farmers)
    ↓
3. Format data into structured prompt:
   "LOCATION: Bangalore, Karnataka
    SOIL: Red Sandy Loam
    PEST ALERTS: Fall Armyworm in Maize (high severity)
    SOURCE: ICAR Pest Surveillance"
    ↓
4. Pass to LLM with instruction:
   "Use ONLY the data provided above"
    ↓
5. LLM generates accurate response:
   "⚠️ Fall Armyworm is currently affecting maize crops 
   in Karnataka. This is a high severity pest. 
   Recommended: Use pheromone traps and IPM methods."
