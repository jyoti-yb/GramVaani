# 🌾 Gram Vaani

> Empowering 700 million rural Indians with voice-first AI in their own language.

Gram Vaani is a full-stack AI platform that lets farmers ask questions by **voice or text** in 9 Indian languages and get instant, spoken answers about weather, crop prices, government schemes, and farming advice available to everyone through Whatsapp and Web Platform — powered by AWS AI services.

## 🧠 The Problem

Rural farmers in India face three critical barriers:
- **Language** — most AI tools only work in English
- **Literacy** — text-based interfaces exclude millions
- **Access** — no single platform for weather , crop prices, Govt schemes, Trusted community alerts

Gram Vaani solves all three with a **voice-first, multilingual, community-powered** platform.

## ✨ Key Features

### 🎤 Voice & Text AI Assistant
- Speak or type in any of 9 Indian languages (Hindi, Tamil, Telugu, Kannada, Malayalam, Bengali, Gujarati, Marathi, English)
- Get AI responses spoken back in the same language
- Powered by **Azure OpenAI GPT-4o-mini** + **Amazon Transcribe** + **Amazon Polly**

### 🌤 Real-Time Weather
- Instant weather for the farmer's registered location
- Farming-specific advice based on current conditions (e.g. "postpone fertilizer spraying")

### 💰 Crop Market Prices
- Live mandi prices for major crops
- Location-aware market data

### 🏛 Government Schemes
- AI-powered lookup for PM-KISAN, PMFBY, subsidies, loans
- Responses in the farmer's preferred language

### 👥 Community Intelligence
- Farmers report pest outbreaks, diseases, and weather events
- Peer validation system — reports verified by 3+ farmers earn a "Verified" badge
- Village Trust Score leaderboard (Gold 🥇 / Silver 🥈 / Bronze 🥉)
- Outbreak map — auto-detects clusters of 5+ reports in the same area

### 📅 Crop Calendar
- Season-aware planting and harvesting schedules (Kharif / Rabi / Summer)
- Translated into the user's language via **Amazon Translate**

### 🤖 Smart Farm Advisor
- AI crop recommendations based on soil parameters (N, P, K, pH, temperature, humidity, rainfall)
- Farming optimization strategies generated by GPT-4o-mini
- Agriculture news feed

### 📱 WhatsApp Integration
- Farmers can interact via WhatsApp without installing any app
- Same AI routing (weather / prices / schemes / general) as the web app


## 🚀 Local Setup

### Prerequisites
- Python 3.11+
- Node.js 18+
- AWS account with DynamoDB, S3, Transcribe, Polly, Translate access
- OpenWeather API key

### 1. Clone the repo
```bash
git clone https://github.com/your-username/gramvaani.git
cd gramvaani
```

### 2. Backend
```bash
cd backend
pip install -r requirements.txt
```

Create a `.env` file in the `backend/` folder using .env.sh

Start the backend:
```bash
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

API docs available at: `http://localhost:8000/docs`

### 3. Frontend
```bash
cd frontend
npm install
```

Create a `.env` file in the `frontend/` folder:
```env
VITE_API_URL=http://localhost:8000
```

Start the frontend:
```bash
npm run dev
```

App runs at: `http://localhost:5173`

## ☁️ AWS Services Used

| Service | Usage |
|---------|-------|
| **Amazon Transcribe** | Converts farmer's voice (WAV/MP3) to text in 9 Indian languages |
| **Amazon Polly** | Converts AI responses to speech (Hindi + English) |
| **Amazon Translate** | Translates crop calendar content to user's language |
| **AWS DynamoDB** | Primary database — users, queries, community reports, village trust |
| **AWS S3** | Temporary audio storage for Transcribe jobs (auto-deleted after use) |
| **AWS App Runner** | Serverless container deployment for the FastAPI backend |

## 🔒 Security

- Passwords hashed with **bcrypt**
- All protected routes require **JWT Bearer token**
- Tokens expire after **30 minutes**
- CORS restricted to known frontend origins
- No credentials stored in code — all via environment variables
